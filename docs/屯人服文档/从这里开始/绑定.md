### 如何在QQ群内绑定自己的账号

#### 为什么必须要绑定账号
- 为了让服务器氛围更好，减少恶意玩家的加入。

#### 如何绑定账号
1. 加入[QQ群：955906383](https://qm.qq.com/q/ZtjcU4SAso)
2. 在QQ群内直接发送
```
/验证 你的ID
```
或者
```
/绑定 你的ID
```
3. 然后应该会立刻绑定成功。

> 例子  
我想要在游戏内使用ID"alazeprt"，那么我就要在群内发送"/绑定 alazeprt"

#### 我发送了正确的ID，为什么没有绑定成功？
- 可能是因为你发送的ID不正确，或者已经绑定过了。
- 你可以尝试再次发送。
- @群内管理员（除了机器人外），看看他们能不能解决。
